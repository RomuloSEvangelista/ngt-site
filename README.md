# 🌐 NewGroup Telecom (NGT) - Landing Page

Este é o projeto de reformulação total da Landing Page institucional da **NewGroup Telecom (NGT)**. O site foi reconstruído do zero com um design moderno e totalmente.


A nova identidade visual foi baseada nas campanhas oficiais da marca, adotando uma paleta de cores energética com degradês em **Laranja**, trazendo muito mais dinamismo, tecnologia e proximidade com o cliente.

---
